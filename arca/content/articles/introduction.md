# Introduction to Arca

Welcome to Arca, a lightweight content management system that stores all data in Markdown files with YAML front matter.

## Why Arca?

Arca was created to provide a simple, file-based approach to content management. Instead of relying on complex databases and APIs, Arca stores everything in plain text files that can be versioned with Git.

## Key Features

1. **File-Based**: All content is stored in Markdown files with YAML metadata
2. **No Database**: No relational database or complex backend - just files
3. **Version Control**: All content can be tracked with Git
4. **Simple Interface**: Clean, fast web interface built with FastHTML and HTMX
5. **Flexible Structure**: Organize content in folders or use metadata references

## Getting Started

To get started with Arca, simply create a new content item using the web interface or by adding a Markdown file to the appropriate directory.
